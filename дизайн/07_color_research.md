# ЦВЕТОВОЙ РЕСЁРЧ: TUMANOV GROUP

## Ключевые находки

### 1. Индустриальный стандарт
- 67%+ лидеров рынка RE используют синий как основной цвет (доверие, стабильность)
- Люксовая недвижимость: монохромные базы (белый/серый/чёрный) + золото/серебро/jewel-tone акценты
- 2026 тренд: уход от "quiet luxury" нейтральности к bold + warm + earthy

### 2. Психология цвета по сегментам
- **Инвестор-рационал:** Синий (доверие) + белый (чистота/прозрачность) + серый (серьёзность)
- **Мечтатель о доме у моря:** Тёплый песочный (комфорт) + океанический синий (мечта) + зелёный (природа, рост)
- **Серийный инвестор:** Чёрный/тёмный (статус) + золотой (эксклюзивность) + минимум цвета
- **Англоязычный инвестор:** Чистый белый (международность) + синий (professional) + тёплые акценты (tropical lifestyle)

### 3. Бенчмарки
- **Whitewill:** Белый минималистичный дизайн, чёрный текст, минимум цвета, лица команды
- **Sotheby's International:** Тёмно-синий (#00205C) + белый + золотистый
- **Knight Frank:** Белый + вишнёво-бордовый акцент
- **Savills:** Белый + красно-оранжевый акцент

### 4. Тропический контекст (Пхукет)
- Лучшие tropical luxury бренды используют: soft blues + sandy beiges + cream + тёплые акценты
- Сочетание warm grounded tones (песок, глина, дерево) с refined modern neutrals
- Баланс между "тропическим ощущением" и "инвестиционной серьёзностью"

## РЕКОМЕНДАЦИЯ

**Направление C: «WARM PROFESSIONAL»** — потому что:
1. Сохраняет Tumanov Blue #0088CC как якорь бренда
2. Тёплый белый фон (#FAFAF7) — не "больничный" белый, но и не тёмный
3. Песочные акценты (#C9A96E) создают tropical premium ощущение
4. Гибкость: один набор цветов, но разные пропорции по каналам
5. Клиенты НЕ будут закрывать PDF — светлый, воздушный, приятный
6. При этом данные и графики читаются легко на белом фоне
7. Соответствует тренду 2026: warm + professional + nature-inspired
