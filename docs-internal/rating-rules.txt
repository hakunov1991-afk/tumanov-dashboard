ПРАВИЛА ПОДСЧЁТА «РЕЙТИНГА»
============================

Полные правила и константы — в CLAUDE.md этого проекта (раздел «Логика Рейтинг»)
и в коде scripts/rating-sync.js (серверный расчёт), docs/js/renderers/rating-period.js
(клиентский селектор периода).

Краткая выжимка:

1. ПЕРИОД
   По умолчанию — последние 3 ПОЛНЫХ месяца (текущий не учитывается).
   Расчёт идёт на клиенте, можно выбрать любой диапазон месяцев в селекторе «От/До».
   База данных rating-db.json содержит месяцы с апреля 2025 по текущий.

2. БРОКЕРЫ
   Источник списка — AMO группа 529606 «Брокеры Пхукет».
   Исключены всегда: «Свободный лид» (12956222), брокер 13781986
   (см. EXCLUDED_BROKER_IDS в scripts/lib/config.js).

3. ВЗЯТО В РАБОТУ
   Переходы лидов на этап 77303674 в выбранном периоде, у которых ответственный — брокер.
   К1-отказы (см. п.5) не учитываются.

4. КРУГ ЛИДА
   Определяется по полям AMO 1630857 (К2), 1630859 (К3), 1631871 (К4)
   на момент takenTs (или closedAt для закрытых сделок 142/143).
   Приоритет: К4 → К3 → К2 → К1.

5. К1-ОТКАЗЫ
   К1 + поле 1617988 заполнено одним из enum'ов
   [7450812, 7450816, 7450818, 7540559, 7555827, 7548773, 7579035]
   → лид полностью исключается из рейтинга.

6. ОТВЕТСТВЕННЫЙ
   - К1 закрытый (142/143) → кто перенёс в финальный статус (created_by закрытия)
   - К1 открытый → текущий responsible_user_id
   - К2/К3/К4 закрытый → кто перенёс в 142/143
   - К2/К3/К4 открытый → исторический ответственный в окне 15 минут после takenTs
     (через события lead_responsible_changed)

7. ПРОШЁЛ MQL
   Из лидов «Взято в работу» — те, что в этом же периоде прошли через этап MQL (77303798).

8. % СЖИГАНИЯ
   (Взято − Прошёл MQL) / Взято × 100%, округлено до целого.

9. ВАЛОВАЯ МАРЖА
   Из Google Sheets таблицы продаж 1S5yUKgOpKGEOHxmYpV0UU7L0UoPE6rNr1BzI-1ekmhw,
   лист «Ответы на форму (1)»: B=дата, C=брокер, CI(87)=маржа,
   AI(35)=ссылка на AMO, DM(117)=2-й брокер, DO(119)=маржа 2-го.
   Сопоставление имени с AMO-брокером — fuzzy (полное → перестановка → фамилия/имя).

10. ЗАТРАТЫ НА MQL
    Стоимость лида по кругу:
      Круг |  До 24.04.2026 |  С 24.04.2026
      К1   |     $100       |     $100
      К2   |     $30        |     $10
      К3   |     $10        |     $10
      К4   |     $0         |     $0
    Применяется по takenTs (момент взятия в работу).

11. ЛИЧНЫЙ ВКЛАД
    Маржа − Затраты на MQL. Округлено до целых. Может быть отрицательным.

12. СОРТИРОВКА И СТАТУС
    DESC по Личному вкладу.
    Место 1 → Лидер, 2 → ТОП 2, 3 → ТОП 3, 4+ и вклад>0 → рентабельный, вклад≤0 → убыточный.

13. КЛИК ПО ЦИФРАМ
    Колонки 3,4,6,7 — кликабельны. Попап со списком AMO-ID и ссылками на сделки.
    Для «Затраты на MQL» — попап разбит по кругам с разделением К2 legacy ($30) и current ($10).

14. ИСКЛЮЧЕНИЯ
    EXCLUDE_LEADS.GLOBAL = ['28672345', '29534441', '29523291', '29416203'] — никогда не считать.
    EXCLUDE_LEADS.BY_MANAGER — конкретные лиды для конкретных брокеров не считать.
    EXCLUDE_LEADS.REASSIGN — принудительно засчитать лид указанному брокеру.
